# LM Studio Interface

Uma interface web simples, minimalista e responsiva para interagir com os seus modelos locais de linguagem rodando no **LM Studio**. 

Este projeto foi desenvolvido utilizando **Vanilla JS (JavaScript Puro), HTML5 e CSS3**, garantindo leveza absoluta, carregamento instantâneo e **zero dependências** de pacotes ou frameworks complexos.

---

## 🚀 Funcionalidades

*   **Interface Minimalista:** Design focado na conversa, sem distrações.
*   **Zero Configuração (No Setup):** Não precisa de `npm install`, Docker ou compilações.
*   **Conexão Direta:** Comunica-se diretamente com a API local do seu LM Studio.
*   **Leve e Veloz:** Carrega instantaneamente em qualquer navegador moderno.

## 🛠️ Tecnologias Utilizadas

O projeto utiliza a stack web nativa:
*   **HTML5:** Estruturação semântica da interface de chat.
*   **CSS3:** Estilização moderna e layout responsivo.
*   **JavaScript (ES6+):** Manipulação assíncrona do DOM e requisições para a API local.
